# 🌟 My OOP Practice Repository

Welcome to my **Object-Oriented Programming** practice repository! Here you'll find various C++ projects and practice exercises focusing on fraction operations, pizza classes, struct usage, and other concepts in OOP.

---

## 📁 Folder Structure
- **fraction projects**: Deals with fraction classes, addition, multiplication, and mixed-number conversions.
- **pizza**: A small class-based program for customizing and displaying pizza details.
- **practice codes**: Various exercises covering arrays, pointers, structs, and class basics.
- **homeworks & classworks**: More structured tasks and assignments exploring OOP techniques.

---

## 🚀 Getting Started
1. **Clone the Repository**  
   ```bash
   git clone https://github.com/YourUsername/YourRepoName.git
   cd YourRepoName
   ```
2. **Open in Your IDE** (e.g., Visual Studio Code).
3. **Build & Run**  
   - For Mac/Linux:
     ```bash
     g++ -std=c++11 filename.cpp -o program
     ./program
     ```
   - For Windows:
     ```bash
     g++ -std=c++11 filename.cpp -o program.exe
     program.exe
     ```
